o site é da semana empresaria de açailandia. é uma semana inteira de eventos: palestras, ferias, cuross, oficinas e feira de exposição. parceiros como governo do maranhão, prefeitura de açailandia, senac,sebrae, senai, fiema e muitos outros parceiros como consta na imagem de parceiros. o principal objeitvo é fortalecer o comercio local e mostrar sua força. 

funcionalidades:
- criar eventos: tipos de eventos, palestas, crusos, oficina, rodada de negocio, feira e etc. 
- fazer inscrição para os eventos
- controle de vagas para os eventos
- agenda dos eventos
- fazer inscrição dos eventos. existem eventos gratuitos e pagos. os pagos vão ser integrados via api asaas. 
- portal de cadastro de eventos, gerenciar eventos, como inscritos, pagamentos e etc.
- criar acessos para parceiros gerenciar seus eventos. por exemplo o sebrae promove seus cursos, fiema tbm os seus, senac tbm. precisam ter acesso para gerenciar seus eventos, acompanhar inscrições e etc.
- dashboar para gerenciamento 
- site com markeplace que mostra todos os eventos gratuitos e pagos com os respectivos filtros. apos escolher o evento vai para o formulario de inscrição e se tiver pagamento realizar o pagamento, apos o pagamento sera redirecionado para a confirmação com o qr code que vai dar accesso aos eventos. a confirmação vai para o email assim como o pdf com o qrcode de acesso.
- pagina para checkin via qr code ou cpf do participante para liberar o acsso aos eventos e confirmar presenã.
- relatorio de presenção para os eventos isso vai mostrar para os stakholderes a presença nos eventos.
- precisa de um sistema de cupon de desconto para associados, combos e etc. ao colocar os codigos vai dar os descontos nos eventos
- na pasta docs tem um banco de dados usado para o evento anterior. voce pode ler para entender quais as informações são necessarias para o formulario de inscrição.
- para a feira de exposição criar um qr code para cada expositor. esse qr code vai ser colado no stand de cada um e o visitante vai scanear o qrcode para concorrer ao sorteio no final da exposição. cada stand que o visitante scanear vai ser mais um numero para ele concorrer. ao final o sistema pode gerar um relatorio de quem visitou o seu stand. ao scanear o qr code seria interessante ter a possibilidade de o visitante adicionar o seu contato. para o dono do stante enviar agradecimentos. voce pode melhorar a ideia.
- o sistema precisa gerar um pdf com o cracha de participante caso seja necessario imprimir e o visitante usar nos eventos para o seu acesso.
- ter cracha para a equipe de staff assim como o cadastro do staff.
- ter uma pagina (landingpage) para cada evento com detalhes do evento. essa opção vai ser o organizador que vai dizer se o evento vai ou não ter landingpage.
